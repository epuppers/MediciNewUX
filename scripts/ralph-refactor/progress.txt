### New Codebase (medici-app/)
- All new code goes inside medici-app/ directory
- Original prototype files remain UNTOUCHED as reference
- Use ~ alias for app-relative imports
- shadcn components installed to app/components/ui/
- Zustand stores in app/stores/
- Service layer in app/services/ wraps mock data behind async functions
- Mock data in app/data/ — this is the ONLY place MOCK_* constants live
- Route files in app/routes/ with loaders for data fetching

### Stack Specifics
- React Router 7 (framework mode): npx create-react-router@latest
- Tailwind v4: @import 'tailwindcss' in CSS, @tailwindcss/vite plugin, @theme for custom values
- shadcn/ui: npx shadcn@latest init detects React Router 7 automatically
- Zustand: create<Type>() for stores, persist() middleware for localStorage
- Dev server: npm run dev (Vite, default port 5173)
- Type check: npx tsc --noEmit
- Build: npm run build
